CSS Introduction

What is CSS?

CSS stands for Cascading Style Sheets.

CSS converts the html page into a beautiful and responsive website.

Types to add css to html

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Inline CSS

Adding style attribute in open tag.

Internal CSS

Adding style tag in body tag

External CSS

Adding a style.css to our html page using link tag in body tag

Selector

It is used to select an html element for styling

        selector{
            property:value;
        }

Example:

        h1{
            color:red;
        }

Here h1 is selector, color is the property and red is the value
